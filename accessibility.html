Accessibility Details

In an effort to make my portfolio website as inclusive as possible, I have taken several different steps:

  • Utilized minimal scripts for optimal loading time.
  • Compressed image formats to accomodate low-bandwidth connections.
  • Alt-text associated with all images to ensure screen reader compatibility.
  • Links to external files (such as PDFs) are labeled accordingly.
  • All links (except those to external files) open in the same tab for back button compatibility.
  • Verified contrast levels through Wave Web Accessibility Tool to ensure ease of reading.
